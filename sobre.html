Sobre o Projeto

Este projeto foi criado por um fã de TBATE chamado Thiago, com o objetivo de comparar fielmente o conteúdo do anime com o manhwa e destacar as mudanças mais significativas. Além de ser um espaço de crítica, é também um exercício de aprendizado em desenvolvimento web.

Tecnologias utilizadas: HTML, Tailwind CSS, JavaScript e GitHub Pages.
